The red outlines and red text show the locations that the system will use for mapping. If we don't have an area for a location, you'll just see a red dot.

Tip: when panning around the map, drag in the middle of an area - that reduces the chance you'll move a marker by mistake.

Changing a Location

  1. Click on the red boundary or the red dot.
  2. The Location Name box will update with the name.
  3. Drag the nodes in the boundary, or use the polygon icon (next to the hand at the top right of the map) to draw a new one.
  4. Click Save Changes.

Deleting a Location

  1. Think carefully. You can't get them back.
  2. Click on the red boundary or the red dot.
  3. The Location Name box will update with the name.
  4. Click Delete Location.

Adding a Location

  1. Drag the map slightly to ensure that you don't have any location currently selected
  2. The Location Name box should be empty.
  3. Enter the name of your location into the location Name box.
  4. Use the polygon icon (next to the hand) to draw a new area.
  5. Click Save Changes.
  6. The new area should appear, unless it doesn't contain any full postcodes.

Renaming a Location

  1. You can't do this for technical reasons.
  2. Delete it and add a new one.
