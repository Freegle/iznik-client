This is work that needs to happen in the background, usually because it needs to communicate with Yahoo.

You can ignore this most of the time, and it's ok to click around within ModTools without worrying about it.

Sometimes you'll see a warning sign - this means ModTools couldn't manage to complete an item of work, but it'll try again later.

If you see the number building up then you need to use this ModTools app when logged in to Yahoo or from a machine with the browser plugin installed.
