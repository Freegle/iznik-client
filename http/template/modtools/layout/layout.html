Please use ModTools regularly from a machine with the plugin installed.

ModTools can only do some things via the browser plugin. Right now you have work like that which has been waiting for at least 4 hours, so some changes won't have been made to Yahoo.

If you already have the plugin installed, see help.

Or if you are on a tablet/mobile, get the IOS/Android app - that has a plugin built-in and works nicely.

You can install the plugin on a desktop/laptop by clicking on the Firefox or Chrome icons on the bottom left. At least one moderator per group needs to use a desktop/laptop or the app regularly. You can just leave a ModTools tab open.
