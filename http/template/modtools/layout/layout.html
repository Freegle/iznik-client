Please use ModTools regularly from a device that is connected to Yahoo.

ModTools can only do some things when connected to Yahoo. Right now you have work like that which has been waiting for at least 4 hours, so some changes won't have been made to Yahoo.

You can install the plugin to connect to Yahoo on a desktop/laptop by clicking on the Firefox or Chrome icons on the bottom left. The ModTools app for Android and iOS can also be used to connect to Yahoo.
